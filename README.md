# 🛒 ApexCart - Modern E-Commerce Web Application

A full-stack, responsive E-commerce web platform built using **Python (Django)**, **HTML5/CSS3**, **JavaScript**, and **Bootstrap 5**. Developed as part of the **CodeAlpha Web Development Internship Task 1**.

---

## ✨ Features

- **Product Showcase & Catalog:** Grid display with prices, badges, search query filtering, and detailed product views.
- **Dynamic Shopping Cart:** Session/Database-linked cart supporting quantity adjustments (`+` / `-`) and item removal.
- **User Authentication:** Complete Register, Login, and Logout functionality powered by Django Auth.
- **Checkout & Order Processing:** Multi-step order summary and Cash on Delivery order processing.
- **Admin Dashboard:** Native Django Admin panel integration to manage products, categories, stock, and customer orders.
- **Modern UI/UX:** Styled with custom Google Fonts (*Plus Jakarta Sans*), micro-interactions, responsive navbar, and glassmorphism styling.

---

## 🛠️ Tech Stack

- **Backend:** Python 3, Django Framework
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap 5, Bootstrap Icons
- **Database:** SQLite3
